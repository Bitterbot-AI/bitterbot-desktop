# `bitterbot setup`

Initialize `~/.bitterbot/bitterbot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
bitterbot setup
bitterbot setup --workspace ~/.bitterbot/workspace
```

To run the wizard via setup:

```bash
bitterbot setup --wizard
```
