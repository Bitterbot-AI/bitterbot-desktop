# `bitterbot tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
bitterbot tui
bitterbot tui --url ws://127.0.0.1:19001 --token <token>
bitterbot tui --session main --deliver
```
